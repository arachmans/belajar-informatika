Array Master

Belajar Array C++ dengan Cara yang Menyenangkan

Kenapa Harus Mempelajari Array?

Array memungkinkan penyimpanan banyak data dengan tipe yang sama dalam satu variabel

Memudahkan pengelolaan data dalam jumlah besar seperti daftar nilai, inventory game, atau data sensor

Dasar untuk struktur data yang lebih kompleks seperti matriks, graf, dan tree

Performa akses data yang cepat dengan indeks langsung

Materi Array

Pengertian Array

Array adalah struktur data yang menyimpan koleksi elemen dengan tipe data yang sama dalam urutan kontigu di memori. Setiap elemen dapat diakses menggunakan indeks numerik.

Cara Kerja Indeks

Indeks digunakan untuk mengakses elemen spesifik dalam array:

{{ val }}

Indeks dimulai dari 0, bukan 1!

Deklarasi Array

C++ 
// Deklarasi array satu dimensi
int angka[5]; // Array dengan 5 elemen integer
double nilai[10]; // Array dengan 10 elemen double
char nama[20]; // Array karakter (string)

Inisialisasi Array

C++ 
// Inisialisasi saat deklarasi
int angka[5] = {10, 20, 30, 40, 50};
char nama[] = "Array Master"; // Otomatis menentukan ukuran

// Inisialisasi sebagian
int nilai[5] = {100, 90}; // Sisanya otomatis 0

Akses Elemen Array

C++ 
// Mengakses elemen (indeks dimulai dari 0)
int arr[5] = {10, 20, 30, 40, 50};

cout << arr[0]; // Output: 10
cout << arr[2]; // Output: 30

// Mengubah nilai elemen
arr[1] = 25; // Mengubah elemen kedua

Mengetahui Jumlah Isi Array

Ada beberapa cara untuk mengetahui jumlah elemen dalam array:

Metode 1: Menggunakan sizeof()

C++ 
int arr[5] = {10, 20, 30, 40, 50};
int jumlahElemen = sizeof(arr) / sizeof(arr[0]);

cout << "Jumlah elemen: " << jumlahElemen; // Output: 5

Metode 2: Untuk array karakter (string)

C++ 
#include <cstring> // Untuk strlen()

char kata[] = "Hello";
int panjang = strlen(kata);

cout << "Panjang string: " << panjang; // Output: 5

Metode 3: Menggunakan konstanta

C++ 
const int UKURAN = 10;
int arr[UKURAN];

for(int i = 0; i < UKURAN; i++) {
    cout << "Elemen ke-" << i << endl;
}

Menggunakan Looping untuk Mengakses Array

Looping sangat berguna untuk mengakses semua elemen array secara efisien:

1. Menampilkan Semua Elemen

C++ 
int nilai[5] = {85, 90, 78, 92, 88};

cout << "Daftar Nilai:" << endl;
for(int i = 0; i < 5; i++) {
    cout << "Nilai ke-" << (i+1) << ": " << nilai[i] << endl;
}
Output:
Daftar Nilai:
Nilai ke-1: 85
Nilai ke-2: 90
Nilai ke-3: 78
Nilai ke-4: 92
Nilai ke-5: 88

2. Menginput Nilai ke Array

C++ 
int data[5];

cout << "Masukkan 5 bilangan:" << endl;
for(int i = 0; i < 5; i++) {
    cout << "Bilangan ke-" << (i+1) << ": ";
    cin >> data[i];
}

cout << "\nData yang dimasukkan:" << endl;
for(int i = 0; i < 5; i++) {

3. Loop dengan sizeof() untuk Array Dinamis

C++ 
int angka[] = {2, 4, 6, 8, 10, 12, 14};

cout << "Genap: ";

4. Loop untuk Mencari Nilai

C++ 
int arr[6] = {23, 45, 12, 67, 34, 89};
int cari = 67;
bool ketemu = false;

for(int i = 0; i < 6; i++) {
    if(arr[i] == cari) {
        cout << cari << " ditemukan di indeks " << i << endl;
        ketemu = true;
        break;
    }
}

if(!ketemu) {
    cout << cari << " tidak ditemukan" << endl;
}

Contoh Penggunaan Array

1. Menghitung Rata-rata Nilai

C++ 
#include <iostream>
using namespace std;

int main() {
    int nilai[5] = {85, 90, 78, 92, 88};
    int total = 0;
    int n = sizeof(nilai) / sizeof(nilai[0]);
    
    for(int i = 0; i < n; i++) {
        total += nilai[i];
    }
    
    double rata = (double)total / n;
    cout << "Rata-rata: " << rata << endl;
    
    return 0;
}
Output:
Rata-rata: 86.6

2. Mencari Nilai Maksimum

C++ 
#include <iostream>
using namespace std;

int main() {
    int data[6] = {23, 45, 12, 67, 34, 89};
    int maks = data[0];
    int n = sizeof(data) / sizeof(data[0]);
    
    for(int i = 1; i < n; i++) {
        if(data[i] > maks) {
            maks = data[i];
        }
    }
    
    cout << "Nilai maksimum: " << maks << endl;
    return 0;
}
Output:
Nilai maksimum: 89

3. Array 2 Dimensi (Matriks)

C++ 
#include <iostream>
using namespace std;

int main() {

        {1, 2, 3},

    return 0;

Output:
1 2 3
4 5 6

4. Program Input dan Output Array Dinamis

C++ 
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Masukkan jumlah elemen: ";
    cin >> n;
    
    int arr[n];
    
    // Input elemen
    cout << "Masukkan " << n << " bilangan:" << endl;
    for(int i = 0; i < n; i++) {
        cout << "Elemen ke-" << (i+1) << ": ";
        cin >> arr[i];
    }
    
    // Menampilkan elemen
    cout << "\nArray yang dimasukkan:" << endl;
    for(int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    
    // Mencari nilai terbesar
    int maks = arr[0];
    for(int i = 1; i < n; i++) {
        if(arr[i] > maks) {
            maks = arr[i];
        }
    }
    cout << "\nNilai terbesar: " << maks << endl;
    
    return 0;
}
Output:
Masukkan jumlah elemen: 4
Masukkan 4 bilangan:
Elemen ke-1: 15
Elemen ke-2: 27
Elemen ke-3: 8
Elemen ke-4: 42

Array yang dimasukkan:
15 27 8 42
Nilai terbesar: 42

Latihan Praktik

Kerjakan soal-soal berikut menggunakan editor kode Anda!

Latihan 1: Kalkulator Array

Soal:

Buat program yang meminta input 5 bilangan bulat dari user, simpan dalam array, kemudian tampilkan:

  • Jumlah total semua bilangan
  • Rata-rata dari semua bilangan
  • Bilangan terbesar dan terkecil

Contoh Output Program:

Masukkan 5 bilangan:
Bilangan 1: 10
Bilangan 2: 25
Bilangan 3: 15
Bilangan 4: 30
Bilangan 5: 20

Hasil:
Jumlah Total: 100
Rata-rata: 20
Terbesar: 30
Terkecil: 10

Latihan 2: Pencarian Elemen

Soal:

Buat program yang:

  • Mendefinisikan array dengan 10 elemen (isi sesuai keinginan)
  • Meminta user memasukkan bilangan yang ingin dicari
  • Mencari bilangan tersebut dalam array
  • Menampilkan posisi indeks jika ditemukan, atau pesan tidak ditemukan

Contoh Output Program:

Array: [5, 12, 7, 3, 9, 15, 2, 8, 11, 6]

Masukkan bilangan yang dicari: 9
Bilangan 9 ditemukan pada indeks 4

Masukkan bilangan yang dicari: 4
Bilangan 4 tidak ditemukan dalam array

Latihan 3: Matriks Penjumlahan

Soal:

Buat program yang:

  • Membuat matriks 3x3 (array 2 dimensi)
  • Meminta user mengisi setiap elemen matriks
  • Menampilkan matriks yang telah diisi
  • Menghitung dan menampilkan jumlah setiap baris dan kolom

Contoh Output Program:

Matriks 3x3:
[1] [2] [3]
[4] [5] [6]
[7] [8] [9]

Jumlah per baris:
Baris 1: 6
Baris 2: 15
Baris 3: 24

Jumlah per kolom:
Kolom 1: 12
Kolom 2: 15
Kolom 3: 18

Tips Pengerjaan:

  • Gunakan loop for untuk mengakses setiap elemen array
  • Ingat bahwa indeks array dimulai dari 0
  • Untuk matriks, gunakan nested loop (loop di dalam loop)
  • Test program Anda dengan berbagai input untuk memastikan berjalan benar
