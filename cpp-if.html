üìã Soal: Sistem Penilaian Siswa

Deskripsi:

Buatlah program C++ yang dapat menentukan grade (nilai huruf) siswa berdasarkan nilai numerik yang diinputkan. Program juga harus memberikan keterangan status kelulusan.

Kriteria Penilaian:

  • Nilai 90-100: Grade A (Lulus dengan Pujian)
  • Nilai 80-89: Grade B (Lulus dengan Baik)
  • Nilai 70-79: Grade C (Lulus)
  • Nilai 60-69: Grade D (Lulus Bersyarat)
  • Nilai 0-59: Grade E (Tidak Lulus)
  • Nilai di luar range 0-100: Input tidak valid

Persyaratan Program:

  1. Gunakan operator perbandingan untuk mengecek range nilai
  2. Gunakan struktur if-else atau switch-case
  3. Tambahkan validasi input (nilai harus 0-100)
  4. Program dapat dijalankan berulang kali
  5. Tampilkan output yang user-friendly

Contoh Output:

// Contoh jalannya program

Masukkan nilai siswa (0-100): 85
Nilai: 85
Grade: B
Status: Lulus dengan Baik

Ingin input nilai lagi? (y/n): n
Terima kasih!

üí° Tips Pengerjaan:

  • Mulai dengan membuat kerangka program (include, main function)
  • Gunakan variabel int untuk menyimpan nilai
  • Implementasikan validasi input terlebih dahulu
  • Gunakan nested if atau if-else if untuk menentukan grade
  • Jangan lupa compile dan test program Anda!
