📖 Materi Hardware Komputer

⌨️ Input Devices (Perangkat Masukan)

Input devices adalah perangkat yang digunakan untuk memasukkan data atau perintah ke dalam komputer.

⌨️

Keyboard

Untuk mengetik huruf dan angka

🖱️

Mouse

Untuk mengklik dan menggerakkan kursor

🎤

Microphone

Untuk merekam suara

📷

Webcam

Untuk mengambil foto dan video

🖥️ Output Devices (Perangkat Keluaran)

Output devices adalah perangkat yang digunakan untuk menampilkan atau mengeluarkan hasil dari komputer.

🖥️

Monitor

Untuk menampilkan gambar dan teks

🔊

Speaker

Untuk mengeluarkan suara

🖨️

Printer

Untuk mencetak dokumen

🎧

Headphone

Untuk mendengar suara pribadi

🧠 Processing Devices (Perangkat Pemroses)

Processing devices adalah perangkat yang memproses data dan menjalankan program komputer.

🔲

CPU

Otak komputer yang memproses data

🎮

GPU

Memproses gambar dan grafik

🧮

RAM

Menyimpan data sementara

🔧

Motherboard

Menghubungkan semua komponen

💿 Storage Devices (Perangkat Penyimpanan)

Storage devices adalah perangkat yang digunakan untuk menyimpan data secara permanen.

💽

Hard Disk

Menyimpan file dan program

⚡

SSD

Penyimpanan cepat tanpa bagian bergerak

🔌

USB Flash Drive

Penyimpanan portabel

💿

CD/DVD

Penyimpanan optik
