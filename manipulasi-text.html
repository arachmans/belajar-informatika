Penggabungan Teks

Satukan beberapa teks menjadi satu kesatuan

CONCAT
CONCATENATE Function

Menggabungkan beberapa teks menjadi satu. Sintaks: CONCATENATE(text1, text2, ...)

=CONCATENATE(A1," ",B1)

💡 Contoh: Jika A1="John" dan B1="Doe"
Hasil: "John Doe"

&
Operator & (Ampersand)

Cara lebih sederhana untuk menggabungkan teks menggunakan simbol &

=A1&" "&B1&" - "&C1

💡 Contoh: Menggabungkan nama dan jabatan
Hasil: "John Doe - Manager"

📏

Menghitung Panjang Teks

Hitung jumlah karakter dalam teks dengan presisi

LEN
LEN Function

Penghitung karakter yang akurat

Menghitung jumlah total karakter dalam teks, termasuk spasi dan simbol.
Sintaks: LEN(text)

=LEN("Excel 2024")
🎯 Contoh Praktis
Input:
"Excel 2024"
↓
Output:
10
Penjelasan: E-x-c-e-l-[spasi]-2-0-2-4 = 10 karakter
💡 Kegunaan LEN
  • • Validasi panjang password
  • • Cek batas karakter tweet/SMS
  • • Analisis data teks
  • • Kontrol kualitas input data
  • • Kombinasi dengan fungsi lain
🔥 Contoh Kombinasi Advanced
Validasi Email:
=IF(LEN(A1)>5,"Valid","Terlalu Pendek")
Ambil Karakter Tengah:
=MID(A1,LEN(A1)/2,1)
Cek Panjang Password:
=LEN(A1)>=8
Hitung Sisa Karakter:
=280-LEN(A1)
✂️

Mengambil Bagian Teks

Ekstrak bagian tertentu dari teks dengan presisi

←
LEFT Function

Mengambil sejumlah karakter dari sebelah kiri teks. Sintaks: LEFT(text, num_chars)

=LEFT("Excel2024",5)

🎯 Contoh Praktis: Mengambil kode produk
Input: "PRD001-Laptop"
Output: "PRD00"

→
RIGHT Function

Mengambil sejumlah karakter dari sebelah kanan teks. Sintaks: RIGHT(text, num_chars)

=RIGHT("Excel2024",4)

🎯 Contoh Praktis: Mengambil tahun dari laporan
Input: "Laporan2024"
Output: "2024"

↔
MID Function

Mengambil karakter dari posisi tertentu. Sintaks: MID(text, start_num, num_chars)

=MID("ID123456789",3,6)

🎯 Contoh Praktis: Mengambil nomor ID
Input: "ID123456789"
Mulai posisi 3, ambil 6 karakter
Output: "123456"

🔤

Mengubah Format Huruf

Kontrol format huruf sesuai kebutuhan

ABC
UPPER

Mengubah semua huruf menjadi huruf kapital/besar

=UPPER("john doe")

📝 Header Laporan
Input: "john doe"
Output: "JOHN DOE"

abc
LOWER

Mengubah semua huruf menjadi huruf kecil

=LOWER("JOHN@EMAIL.COM")

📧 Format Email
Input: "JOHN@EMAIL.COM"
Output: "john@email.com"

Abc
PROPER

Mengubah huruf pertama setiap kata menjadi kapital (Title Case)

=PROPER("john doe smith")

👤 Nama Proper
Input: "john doe smith"
Output: "John Doe Smith"

🧹

Membersihkan Spasi

Rapikan data dengan menghilangkan spasi berlebih

TRIM
TRIM Function

Pembersih spasi yang powerful

Menghapus spasi berlebih di awal, akhir, dan tengah teks (menyisakan 1 spasi antar kata).
Sintaks: TRIM(text)

=TRIM(" John Doe ")
🎯 Contoh Praktis
Input:
" John Doe "
↓
Output:
"John Doe"
💡 Tips Pro
  • • Sangat berguna untuk data copy-paste
  • • Membersihkan data import dari sistem lain
  • • Merapikan input form yang tidak konsisten
  • • Persiapan data sebelum analisis
🔍

Mencari Posisi Teks

Temukan lokasi karakter atau kata dalam teks

🎯
FIND Function

Mencari posisi karakter/teks tertentu (case-sensitive). Sintaks: FIND(find_text, within_text, start_num)

=FIND("@","john.doe@company.com")

📧 Memisahkan Email
Cari posisi @ untuk split username/domain
Input: "john.doe@company.com"
Output: posisi 9

⚠️ Case-Sensitive: Membedakan huruf besar/kecil

🔎
SEARCH Function

Sama seperti FIND tapi tidak case-sensitive dan mendukung wildcard. Sintaks: SEARCH(find_text, within_text, start_num)

=SEARCH("excel","Microsoft Excel 2024")

📝 Pencarian Fleksibel
Cari kata tanpa peduli besar/kecil
Input: "Microsoft Excel 2024"
Output: posisi 11

✅ Case-Insensitive: Tidak membedakan huruf besar/kecil

🔄

Mengganti Teks

Ubah atau ganti bagian teks dengan mudah

🎯
SUBSTITUTE

Mengganti teks tertentu dengan teks baru. Sintaks: SUBSTITUTE(text, old_text, new_text, instance_num)

=SUBSTITUTE("081-234-567-890","-","")

📱 Format Nomor Telepon
Menghapus tanda hubung
Input: "081-234-567-890"
Output: "081234567890"

💡 Tip: Bisa mengganti semua kemunculan atau hanya yang ke-n

📍
REPLACE

Mengganti karakter pada posisi tertentu. Sintaks: REPLACE(old_text, start_num, num_chars, new_text)

=REPLACE("ID001234",3,3,"999")

🆔 Update Kode ID
Ganti 3 karakter mulai posisi 3
Input: "ID001234"
Output: "ID999234"

⚡ Presisi: Ganti berdasarkan posisi, bukan isi teks

🤔 Kapan Menggunakan Mana?
SUBSTITUTE:
  • • Ketika tahu teks yang ingin diganti
  • • Menghapus karakter tertentu
  • • Standardisasi format data
REPLACE:
  • • Ketika tahu posisi yang ingin diganti
  • • Update bagian tertentu dari kode
  • • Ganti karakter di posisi tetap

📋 Skenario:

Anda bekerja di departemen HR dan memiliki data karyawan yang perlu dirapikan. Data berisi nama lengkap, email, dan nomor telepon yang formatnya tidak konsisten.

📊 Data Mentah:

ID Karyawan Nama Lengkap Email Telepon

EMP002567 jane smith JANE.SMITH@COMPANY.COM 08123456789
EMP003890 Bob Johnson bob.johnson@company.com +62 812 3456 7891
MGR004123 ALICE BROWN alice.brown@company.com +62-813-4567-8901
EMP005456 charlie davis CHARLIE.DAVIS@COMPANY.COM 081345678902
SUP006789 EVA WILSON eva.wilson@company.com +62 814 5678 9012
EMP007012 frank miller frank.miller@company.com +62-815-6789-0123
MGR008345 GRACE TAYLOR GRACE.TAYLOR@COMPANY.COM 081567890123
EMP009678 henry anderson henry.anderson@company.com +62 816 7890 1234
SUP010901 IVY THOMAS ivy.thomas@company.com +62-817-8901-2345

🎯 Tugas Anda:

  1. Rapikan nama (hilangkan spasi berlebih, gunakan Title Case)
  2. Buat kolom "Nama Depan" dan "Nama Belakang"
  3. Standardkan format email (huruf kecil semua)
  4. Buat kolom "Domain Email" (ambil bagian setelah @)
  5. Standardkan format nomor telepon (format: +62-XXX-XXXX-XXXX)
  6. Buat kolom "Inisial" (huruf pertama nama depan + belakang)
  7. Buat kolom "Jabatan" berdasarkan ID: Jika 3 karakter pertama ID adalah "MGR" maka "Manager", jika "SUP" maka "Supervisor", selain itu "Employee"
  8. Buat kolom "Nomor Urut": Ambil 3 digit terakhir dari ID Karyawan

💡 Rumus yang Dibutuhkan:

TRIM & PROPER: Rapikan nama
LEFT & FIND: Ambil nama depan
RIGHT & LEN: Ambil nama belakang
LOWER: Standardkan email
MID & FIND: Ambil domain
SUBSTITUTE: Format telepon
IF & LEFT: Tentukan jabatan
RIGHT: Ambil nomor urut
