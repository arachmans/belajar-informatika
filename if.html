1. Konsep Dasar & Lanjutan Fungsi IF

Fungsi IF adalah fondasi pengambilan keputusan di Excel. Ia menguji kondisi dan memberikan hasil berbeda tergantung apakah kondisi itu Benar (TRUE) atau Salah (FALSE).

Struktur Rumus Dasar

=IF(logical_test; value_if_true; value_if_false)

Contoh Sederhana

Kita ingin menentukan status kelulusan siswa. Jika nilai di sel A2 lebih besar atau sama dengan 75, maka siswa "Lulus". Jika tidak, maka "Remedial".

=IF(A2>=75; "Lulus"; "Remedial")

IF Bertingkat (Nested IF)

Bagaimana jika kita punya lebih dari dua kemungkinan hasil? Misalnya, predikat nilai A, B, C, D. Di sinilah IF Bertingkat berperan. Kita menempatkan fungsi IF di dalam argumen value_if_false dari fungsi IF lainnya.

Contoh: Mengubah nilai di sel A2 menjadi predikat.

=IF(A2>=90; "A"; IF(A2>=80; "B"; IF(A2>=70; "C"; "D")))

Excel akan mengecek kondisi pertama (>=90). Jika TRUE, hasilnya "A" dan berhenti. Jika FALSE, ia lanjut ke IF kedua (>=80), dan seterusnya.

Kombinasi dengan AND dan OR

Terkadang, satu kondisi saja tidak cukup. Kita perlu menguji beberapa kondisi sekaligus. Untuk ini, kita gunakan AND dan OR di dalam logical_test.

  • Fungsi AND: Menghasilkan TRUE hanya jika SEMUA kondisi di dalamnya terpenuhi.

    Sintaksis: AND(kondisi1; kondisi2; ...)

    Contoh: Memberi bonus jika penjualan (C2) di atas target (B2) DAN masa kerja (D2) lebih dari 1 tahun.

    =IF(AND(C2>=B2; D2>12); "Dapat Bonus"; "Tidak Dapat")
  • Fungsi OR: Menghasilkan TRUE jika SALAH SATU kondisi di dalamnya terpenuhi.

    Sintaksis: OR(kondisi1; kondisi2; ...)

    Contoh: Memberi diskon jika pelanggan adalah member (E2="Ya") ATAU total belanja (F2) di atas 500.000.

    =IF(OR(E2="Ya"; F2>500000); "Dapat Diskon"; "Tidak Dapat")

2. Asah Kemampuan (10 Soal)

1. Sel A1 berisi angka 85. Apa hasil dari =IF(A1>80; "Istimewa"; "Biasa")?

2. Bagian mana yang disebut logical_test dari rumus =IF(B5<100; "Diskon"; "Normal")?

3. Tulis rumus: Jika stok (C2) < 10, tampilkan "Segera Pesan", jika tidak "Stok Aman".

4. Suhu (D10) adalah 25. Hasil dari =IF(D10>30; "Panas"; IF(D10<18; "Dingin"; "Sejuk"))?

5. Penjualan (E5) adalah 6.000.000. Berapa bonus dari =IF(E5>5000000; E5*10%; 0)?

6. Tulis rumus predikat untuk nilai di A5: Jika >=85 "Baik", jika >=70 "Cukup", selain itu "Kurang".

7. Tulis rumus: Jika Kategori (B3) adalah "Buah" DAN Stok (C3) > 100, hasilnya "Promosi". Jika tidak, "Normal".

8. A1="Lulus", B1=90. Hasil dari =IF(AND(A1="Lulus"; B1>85); "Beasiswa"; "-")?

9. A1="Pending", B1="Lunas". Hasil dari =IF(OR(A1="Lunas"; B1="Lunas"); "OK"; "Tahan")?

10. Tulis rumus: Jika Jabatan (C4) "Manajer" ATAU Masa Kerja (D4) > 60 bulan, dapat "Cuti Ekstra". Jika tidak, "Standar".

3. Studi Kasus: Komisi Penjualan

Anda adalah seorang manajer penjualan. Tugas Anda adalah membuat sistem perhitungan insentif untuk tim Anda di Excel. Buatlah tabel berikut:

Tugas Anda:

Isilah kolom-kolom berwarna biru muda dengan rumus Excel berdasarkan ketentuan di bawah ini.

1. Status Target

Jika Penjualan Aktual >= Target, hasilnya "Tercapai". Jika tidak, "Tidak Tercapai".

2. Besar Komisi (IF Bercabang)

  • Jika Status Target "Tercapai", komisi 10% dari Aktual.
  • Jika tidak, cek: apakah Aktual > 80% dari Target? Jika iya, komisi 2% dari Aktual.
  • Jika tidak, komisi 0.

3. Bonus Tambahan (IF + AND)

Bonus Rp 500.000 jika Status Target "Tercapai" DAN Lama Kerja > 12 bulan. Jika tidak, 0.
