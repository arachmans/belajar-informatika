📚 Memahami Istilah Dasar Pemrograman

Mari kenali istilah-istilah penting dalam dunia pemrograman

🖥️ Program

Kumpulan instruksi yang ditulis untuk komputer agar melakukan tugas tertentu.

Analogi: Seperti resep masakan - langkah demi langkah untuk membuat nasi goreng!

⚡ Pemrograman

Proses menulis, menguji, dan memelihara kode program.

Analogi: Seperti menulis skenario film - kamu yang menentukan alur ceritanya!

🗣️ Bahasa Pemrograman

Bahasa khusus yang digunakan untuk berkomunikasi dengan komputer.

Contoh: C++, Python, Java - seperti bahasa Inggris, Indonesia, Jepang untuk manusia!

🎯 Fungsi Bahasa Pemrograman

  • • Memberikan instruksi ke komputer
  • • Memecahkan masalah secara sistematis
  • • Membuat aplikasi dan sistem
  • • Mengotomatisasi pekerjaan

🏗️ Tingkat Bahasa Pemrograman

Dari yang paling dekat dengan mesin hingga yang mudah dipahami manusia

🔧 Bahasa Tingkat Rendah (Low Level)

Bahasa yang sangat dekat dengan bahasa mesin, sulit dipahami manusia tapi sangat efisien.

Assembly Language

MOV AX, 5
ADD AX, 3
MOV BX, AX

Kegunaan:

  • • Sistem operasi
  • • Driver hardware
  • • Embedded systems

⚖️ Bahasa Tingkat Menengah (Middle Level)

Kombinasi antara efisiensi dan kemudahan, cocok untuk berbagai keperluan.

Contoh: C, C++

#include <iostream>
int main() {
  cout << "Hello World!";
  return 0;
}

Kegunaan:

  • • Game development
  • • Aplikasi desktop
  • • Sistem database

🚀 Bahasa Tingkat Tinggi (High Level)

Bahasa yang mudah dipahami manusia, mirip bahasa Inggris sehari-hari.

Contoh: Python, JavaScript

print("Hello World!")
nama = "Budi"
umur = 17
print(f"Hai {nama}, umur {umur}")

Kegunaan:

  • • Website dan aplikasi web
  • • Data science
  • • Artificial Intelligence

🎭 Paradigma Pemrograman

Berbagai gaya dan pendekatan dalam menulis program

📋 Pemrograman Prosedural

Program ditulis sebagai urutan langkah-langkah (seperti resep masakan)

Contoh Bahasa: C, Pascal

1. Mulai
2. Input nilai
3. Hitung hasil
4. Tampilkan hasil
5. Selesai

Analogi: Seperti mengikuti panduan merakit LEGO step by step!

🏗️ Pemrograman Berorientasi Objek

Program dibuat dari objek-objek yang saling berinteraksi

Contoh Bahasa: Java, C++, Python

Objek Siswa:
- Nama, Kelas, Nilai
- Belajar(), Ujian(), Lulus()

Analogi: Seperti bermain The Sims - setiap karakter punya sifat dan kemampuan!

🔄 Pemrograman Fungsional

Program dibuat dari fungsi-fungsi matematika

Contoh Bahasa: Haskell, Lisp

f(x) = x + 1
g(x) = x * 2
hasil = g(f(5))

Analogi: Seperti kalkulator scientific - input masuk, hasil keluar!

⚡ Pemrograman Event-Driven

Program merespons kejadian/event (klik, input, dll)

Contoh Bahasa: JavaScript, Visual Basic

Ketika tombol diklik → Tampilkan pesan
Ketika mouse hover → Ubah warna
Ketika form disubmit → Validasi data

Analogi: Seperti remote TV - tekan tombol, ada reaksi!

🔄 Compiler vs Interpreter

Dua cara berbeda untuk menjalankan program

⚡

Compiler

📝 Cara Kerja:

Menerjemahkan SELURUH kode sekaligus menjadi bahasa mesin sebelum dijalankan

🏃‍♂️ Kecepatan:

Sangat cepat saat dijalankan karena sudah dalam bahasa mesin

🔍 Deteksi Error:

Menemukan semua error sebelum program dijalankan

📱 Contoh Bahasa:

C, C++, Rust, Go

🎯 Analogi: Seperti menerjemahkan buku dari bahasa Inggris ke Indonesia secara lengkap, baru kemudian dibaca!

🔄

Interpreter

📝 Cara Kerja:

Menerjemahkan dan menjalankan kode BARIS DEMI BARIS secara langsung

🏃‍♂️ Kecepatan:

Lebih lambat karena harus menerjemahkan sambil jalan

🔍 Deteksi Error:

Error ditemukan saat baris tersebut dijalankan

📱 Contoh Bahasa:

Python, JavaScript, PHP

🎯 Analogi: Seperti penerjemah simultan yang menerjemahkan sambil pembicara berbicara!

🤔 Mana yang Lebih Baik?

Pilih Compiler jika:

  • • Butuh performa tinggi (game, sistem)
  • • Program akan didistribusi ke banyak user
  • • Ingin melindungi source code

Pilih Interpreter jika:

  • • Ingin development yang cepat
  • • Sering testing dan debugging
  • • Membuat prototype atau script

👋 Berkenalan dengan C++

Bahasa pemrograman yang powerful dan populer di dunia

🚀 Apa itu C++?

C++ adalah bahasa pemrograman yang dikembangkan dari bahasa C oleh Bjarne Stroustrup pada tahun 1985. Bahasa ini menggabungkan kekuatan bahasa tingkat rendah dengan kemudahan pemrograman berorientasi objek.

Contoh Program C++ Sederhana:

#include <iostream>
using namespace std;

int main() {
    cout << "Halo, Siswa SMA!" << endl;
    cout << "Selamat belajar C++!" << endl;
    return 0;
}

Fun Fact: Nama C++ berasal dari operator increment (++) dalam bahasa C, yang berarti "C yang ditingkatkan"!
