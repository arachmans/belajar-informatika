Apa itu Tabel di HTML?

Bayangkan sebuah spreadsheet seperti Microsoft Excel atau Google Sheets. Tabel di HTML punya fungsi yang sama: menampilkan data dalam format baris dan kolom. Ini sangat berguna untuk menyajikan data terstruktur seperti jadwal, daftar harga, atau data statistik.

Struktur dasar sebuah tabel terdiri dari 3 bagian utama:

  • <table>: Tag pembungkus yang menandakan kita akan membuat sebuah tabel.
  • <tr>: Singkatan dari "Table Row", digunakan untuk membuat baris.
  • <td>: Singkatan dari "Table Data", digunakan untuk membuat sel data di dalam setiap baris.

Mari kita lihat contoh paling sederhana:

<table>
  <tr>
    <td>Baris 1, Kolom 1</td>
    <td>Baris 1, Kolom 2</td>
  </tr>
  <tr>
    <td>Baris 2, Kolom 1</td>
    <td>Baris 2, Kolom 2</td>
  </tr>
</table>

Hasilnya akan terlihat seperti ini (saya tambahkan garis bantu agar terlihat jelas):

Baris 1, Kolom 1 Baris 1, Kolom 2
Baris 2, Kolom 1 Baris 2, Kolom 2

Mudah, bukan? Di bagian selanjutnya, kita akan pelajari tag-tag lain dan atributnya.

Tag HTML Penting dan Atributnya

Selain tag dasar, ada beberapa tag dan atribut lain yang sangat penting untuk membuat tabel yang informatif.

Tag <th> (Table Header)

Tag ini digunakan untuk membuat sel header atau judul kolom/baris. Secara default, teks di dalam <th> akan tercetak tebal dan berada di tengah. Ini sangat baik untuk aksesibilitas dan SEO.

<table>
  <tr>
    <th>Nama</th>
    <th>Pekerjaan</th>
  </tr>
  <tr>
    <td>Budi</td>
    <td>Programmer</td>
  </tr>
</table>

Hasil:

Nama Pekerjaan
Budi Programmer

Atribut colspan (Column Span)

Atribut ini digunakan untuk menggabungkan beberapa kolom menjadi satu. Nilainya adalah jumlah kolom yang ingin digabungkan.

<table>
  <tr>
    <th colspan="2">Informasi Pribadi</th>
  </tr>
  <tr>
    <td>Budi</td>
    <td>Programmer</td>
  </tr>
</table>

Pada contoh di atas, sel "Informasi Pribadi" akan memanjang selebar 2 kolom.

Hasil:

Informasi Pribadi
Budi Programmer

Atribut rowspan (Row Span)

Mirip seperti colspan, tapi atribut ini digunakan untuk menggabungkan beberapa baris menjadi satu.

<table>
  <tr>
    <th>Nama</th>
    <td>Budi</td>
  </tr>
  <tr>
    <th rowspan="2">Telepon</th>
    <td>081234567</td>
  </tr>
  <tr>
    <td>087654321</td>
  </tr>
</table>

Pada contoh di atas, sel "Telepon" akan memanjang ke bawah setinggi 2 baris.

Hasil:

Nama Budi
Telepon 081234567
087654321

Struktur Semantik: <thead>, <tbody>, dan <tfoot>

Untuk tabel yang kompleks, kita bisa mengelompokkan baris-baris menjadi tiga bagian:

  • <thead>: Untuk bagian kepala/header tabel.
  • <tbody>: Untuk bagian isi/tubuh utama tabel.
  • <tfoot>: Untuk bagian kaki/footer tabel (misalnya total).

Ini tidak mengubah tampilan visual secara default, tapi sangat membantu struktur kode dan aksesibilitas.

Menghias Tabel dengan CSS

Tabel HTML standar terlihat sangat polos. Dengan CSS (Cascading Style Sheets), kita bisa membuatnya jauh lebih menarik dan mudah dibaca.

border & border-collapse

Properti paling dasar adalah garis (border). Agar garis antar sel tidak dobel, kita gunakan border-collapse: collapse;. Properti width: 100%; juga ditambahkan agar tabel memenuhi lebar kontainer.

/* CSS */
.border-table table, .border-table th, .border-table td {
  border: 1px solid black;
}
.border-table table {
  border-collapse: collapse;
  width: 100%;
}

Produk Harga
Buku Rp 50.000

padding

Memberi ruang di dalam sel agar teks tidak menempel pada garis. Ini membuat tabel lebih rapi.

/* CSS */
.padding-table th, .padding-table td {
  padding: 12px;
}

Hasil:

Produk Harga
Buku Rp 50.000

text-align

Mengatur perataan teks di dalam sel (left, right, center). Di sini, header rata tengah dan data rata kanan.

/* CSS */
.align-table th {
  text-align: center;
}
.align-table td {
  text-align: right;
}

Produk Harga
Buku Rp 50.000

Warna Latar (Zebra Stripes)

Membuat warna baris selang-seling (zebra-striping) sangat membantu keterbacaan. Kita bisa menggunakan pseudo-class :nth-child().

/* CSS */
.zebra-table thead {
  background-color: #c084fc;
  color: white;
}
/* baris ganjil */
.zebra-tabel tbody tr:nth-child(odd) {
    background-color: #C28AFF;
}
/* baris genap */
.zebra-table tbody tr:nth-child(even) {
  background-color: #f3e8ff;
}
.zebra-table tbody tr:hover {
    background-color: #d8b4fe;
}

Dengan properti-properti di atas, tabel yang tadinya polos bisa menjadi jauh lebih profesional.
