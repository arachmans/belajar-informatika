Hardware (Perangkat Keras)

Segala komponen fisik dari sistem komputer yang dapat dilihat dan disentuh. Klik tab di bawah untuk melihat jenis-jenisnya.

Perangkat Input

Alat untuk memasukkan data atau perintah ke dalam komputer.

  • Keyboard
  • Mouse
  • Scanner
  • Mikrofon
  • Webcam

Perangkat Proses

"Otak" komputer yang memproses semua instruksi.

  • CPU (Central Processing Unit)
  • Motherboard
  • RAM (Random Access Memory)

Perangkat Output

Alat untuk menampilkan hasil dari data yang telah diproses.

  • Monitor
  • Printer
  • Speaker
  • Proyektor

Perangkat Penyimpanan

Alat untuk menyimpan data secara permanen atau sementara.

  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • Flashdisk

Software (Perangkat Lunak)

Kumpulan instruksi elektronik yang dijalankan komputer untuk melakukan fungsi tertentu. Software tidak memiliki bentuk fisik.

Sistem Operasi (OS)

Software paling dasar yang mengelola semua sumber daya komputer. Tanpanya, komputer tidak bisa digunakan.

  • Microsoft Windows
  • macOS
  • Linux
  • Android & iOS

Software Aplikasi

Program yang dirancang untuk membantu pengguna melakukan tugas spesifik.

  • Microsoft Office (Word, Excel)
  • Google Chrome
  • Adobe Photoshop
  • Game

Brainware (Pengguna)

Manusia yang terlibat dalam penggunaan, pengoperasian, serta pengaturan sistem komputer. Brainware adalah "otak" di balik teknologi.

Programmer

Pembuat software.

System Analyst

Perancang sistem.

Operator

Pengguna operasional.

Pengguna Akhir

Kita semua!
