Materi Looping

Kenapa Harus Mempelajari Looping?

Looping adalah salah satu konsep paling fundamental dalam pemrograman. Tanpa looping, kita harus menulis kode yang sama berulang kali secara manual. Bayangkan harus menulis cout << "Hello"; sebanyak 1000 kali!

Efisiensi

Mengulangi tugas tanpa menulis ulang kode. 1 baris loop bisa menggantikan 1000 baris kode manual!

Otomasi

Menjalankan proses berulang secara otomatis seperti membaca file, memproses data, atau animasi.

Skalabilitas

Program dapat menangani data dalam jumlah besar tanpa perlu modifikasi kode.

Contoh Dunia Nyata:

Game Development

Animasi karakter, update posisi, deteksi tabrakan

Data Processing

Menganalisis ribuan baris data spreadsheet

AI & Machine Learning

Training model dengan jutaan data

Web Development

Menampilkan produk dari database

1 Perulangan For

Perulangan for digunakan ketika kita tahu berapa kali perulangan akan dilakukan.

Visualisasi For Loop

1 Inisialisasi: int i = 1
2 Cek Kondisi: i <= 5? (Benar)
3 Eksekusi Kode: cout << i
4 Increment: i++
5 Ulangi dari langkah 2
C++ 
#include <iostream>
using namespace std;

int main() {
    // Menampilkan angka 1 hingga 5
    for (int i = 1; i <= 5; i++) {
        cout << "Angka: " << i << endl;
    }
    return 0;
}

Output:

Angka: 1
Angka: 2
Angka: 3
Angka: 4
Angka: 5

2 Perulangan While

Perulangan while digunakan ketika kondisi berhenti diketahui.

Visualisasi While Loop

1 Inisialisasi: int i = 1
2 Cek Kondisi: while(i <= 5)
3 Jika benar, eksekusi kode di dalam
4 Update variabel: i++
5 Kembali ke cek kondisi
C++ 
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    // Menampilkan angka 1 hingga 5
    while (i <= 5) {
        cout << "Angka: " << i << endl;
        i++; // Increment
    }
    return 0;
}

Output:

Angka: 1
Angka: 2
Angka: 3
Angka: 4
Angka: 5

3 Perulangan Do-While

Perulangan do-while selalu dijalankan minimal satu kali.

Visualisasi Do-While Loop

1 Jalankan kode dulu (minimal 1x)
2 Cek kondisi: while(i <= 5)
3 Jika benar, ulang dari awal
4 Jika salah, keluar dari loop
C++ 
#include <iostream>
using namespace std;

int main() {
    int i = 1;
    // Menampilkan angka 1 hingga 5
    do {
        cout << "Angka: " << i << endl;
        i++; // Increment
    } while (i <= 5);
    return 0;
}

Output:

Angka: 1
Angka: 2
Angka: 3
Angka: 4
Angka: 5

Perbandingan Visual: For vs While vs Do-While

For Loop

Start → Cek → Run → Inc

✓ Jumlah iterasi diketahui

✓ Semua kontrol di satu baris

✓ Bisa 0 iterasi

While Loop

Cek → Run → Update

✓ Kondisi kompleks

✓ Fleksibel

✓ Bisa 0 iterasi

Do-While Loop

Run → Cek → Ulang

✓ Minimal 1 iterasi

✓ Input validation

✓ Menu systems

VS

Kapan menggunakan?

For: Array, Counter While: File read, Search Do-While: Menu, Input

Perbandingan Ketiga Jenis Looping

Jenis Kapan Digunakan Kelebihan
For Jumlah iterasi diketahui Struktur terorganisir, semua kontrol dalam satu baris
While Iterasi berdasarkan kondisi, bisa nol kali Fleksibel, cocok untuk kondisi kompleks
Do-While Minimal satu iterasi diperlukan Memastikan kode dijalankan setidaknya sekali

Latihan Praktik

Soal 1: Menampilkan Pola Bintang

Buat program yang menampilkan pola bintang segitiga siku-siku dengan tinggi 5 menggunakan perulangan for.

Contoh Output yang Diharapkan:

*
**
***
****
*****

Contoh Solusi:

C++ 
#include <iostream>
using namespace std;

int main() {
    // Perulangan luar untuk baris
    for (int i = 1; i <= 5; i++) {
        // Perulangan dalam untuk bintang
        for (int j = 1; j <= i; j++) {
            cout << "*";
        }
        cout << endl; // Pindah baris
    }
    return 0;
}

Cara Kerja Program:

  • Perulangan luar (variabel i) mengontrol jumlah baris (1 hingga 5)
  • Perulangan dalam (variabel j) mengontrol jumlah bintang di setiap baris
  • Untuk setiap baris ke-i, program mencetak i bintang
  • Setelah selesai mencetak bintang di satu baris, program pindah ke baris baru

Soal 2: Kalkulator Faktorial

Buat program yang menghitung faktorial dari sebuah bilangan bulat positif menggunakan perulangan while.

Contoh Output yang Diharapkan:

Masukkan bilangan: 5
Faktorial dari 5 adalah: 120

Contoh Solusi:

C++ 
#include <iostream>
using namespace std;

int main() {
    int num, factorial = 1;
    
    cout << "Masukkan bilangan: ";
    cin >> num;
    
    int i = num;
    while (i > 0) {
        factorial *= i;
        i--;
    }
    
    cout << "Faktorial dari " << num << " adalah: " << factorial << endl;
    
    return 0;
}

Cara Kerja Program:

  • Program meminta pengguna memasukkan sebuah bilangan
  • Program menginisialisasi variabel factorial dengan nilai 1
  • Perulangan while dimulai dari bilangan yang dimasukkan hingga mencapai 0
  • Di setiap iterasi, nilai factorial dikalikan dengan nilai i saat ini
  • Nilai i dikurangi 1 di setiap iterasi
  • Setelah perulangan selesai, program menampilkan hasil faktorial

Tip: Coba jalankan kode-kode di atas di compiler C++ favorit Anda untuk melihat hasilnya secara langsung!
