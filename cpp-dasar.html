ü§î Apa itu Variabel?

Variabel adalah tempat penyimpanan data dalam memori komputer yang memiliki nama dan dapat berubah nilainya selama program berjalan. Bayangkan variabel seperti kotak yang diberi label untuk menyimpan berbagai jenis barang.

üé® Tipe Data Dasar

üî¢ int

Bilangan bulat

Contoh: 10, -5, 0
üìä float/double

Bilangan desimal

Contoh: 3.14, -2.5
üî§ char

Karakter tunggal

Contoh: 'A', 'x', '5'
üìù string

Teks/kalimat

Contoh: "Hello World"
‚úÖ bool

Nilai benar/salah

Contoh: true, false

‚ö° Cara Deklarasi Variabel 

// Format: tipe_data nama_variabel;
int umur;
float tinggi;
char grade;
string nama;
bool lulus;

// Deklarasi dengan inisialisasi
int umur = 20;
float tinggi = 175.5;
char grade = 'A';
string nama = "Budi";
bool lulus = true;

üîí Variabel Konstanta

Konstanta adalah variabel yang nilainya tidak dapat diubah setelah dideklarasikan. Gunakan kata kunci const untuk membuat konstanta.

‚ö†Ô∏è Aturan Konstanta
  • Harus diinisialisasi saat deklarasi
  • Nilai tidak dapat diubah setelah deklarasi
  • Biasanya ditulis dengan huruf KAPITAL
  • Membantu mencegah kesalahan dalam program
// Deklarasi konstanta
const int MAX_SISWA = 30;
const float PI = 3.14159;
const string NAMA_SEKOLAH = "SMA Negeri 1";

// Konstanta harus diinisialisasi
const int NILAI_LULUS = 75;

// ERROR: Tidak bisa mengubah nilai konstanta
// MAX_SISWA = 35; // Ini akan error!

üöÄ Contoh Penggunaan Lengkap 

#include <iostream>
using namespace std;

int main() {
    // Deklarasi konstanta
    const float PI = 3.14159;
    const int NILAI_LULUS = 75;
    
    // Deklarasi variabel
    string nama = "Alice";
    int umur = 25;
    float ipk = 3.75;
    bool lulus = true;
    float jari_jari = 7.0;
    
    // Menghitung luas lingkaran menggunakan konstanta
    float luas = PI * jari_jari * jari_jari;
    
    // Menampilkan nilai variabel
    cout << "Nama: " << nama << endl;
    cout << "Umur: " << umur << " tahun" << endl;
    cout << "IPK: " << ipk << endl;
    cout << "Status: " << (lulus ? "Lulus" : "Tidak Lulus") << endl;
    cout << "Luas lingkaran: " << luas << endl;
    
    return 0;
}

üéØ Contoh Program Lengkap 

#include <iostream>
using namespace std;

int main() {
    int a = 15, b = 4;
    int x = 10;
    
    // Operator Aritmatika
    cout << "=== OPERATOR ARITMATIKA ===" << endl;
    cout << "a = " << a << ", b = " << b << endl;
    cout << "a + b = " << (a + b) << endl;  // 19
    cout << "a - b = " << (a - b) << endl;  // 11
    cout << "a * b = " << (a * b) << endl;  // 60
    cout << "a / b = " << (a / b) << endl;  // 3
    cout << "a % b = " << (a % b) << endl;  // 3
    
    // Increment dan Decrement
    cout << "\n=== INCREMENT & DECREMENT ===" << endl;
    cout << "a++ = " << a++ << endl;      // 15, lalu a menjadi 16
    cout << "++b = " << ++b << endl;      // 5
    cout << "--a = " << --a << endl;      // 15
    cout << "b-- = " << b-- << endl;      // 5, lalu b menjadi 4
    
    // Operator Assignment
    cout << "\n=== OPERATOR ASSIGNMENT ===" << endl;
    cout << "x awal = " << x << endl;    // 10
    
    x += 5;  // x = x + 5
    cout << "x += 5, x = " << x << endl;  // 15
    
    x -= 3;  // x = x - 3
    cout << "x -= 3, x = " << x << endl;  // 12
    
    x *= 2;  // x = x * 2
    cout << "x *= 2, x = " << x << endl;  // 24
    
    x /= 4;  // x = x / 4
    cout << "x /= 4, x = " << x << endl;  // 6
    
    x %= 4;  // x = x % 4
    cout << "x %= 4, x = " << x << endl;  // 2
    
    return 0;
}

üßÆ Soal 1: Kalkulator Sederhana

üìã Deskripsi Tugas

Buatlah program kalkulator sederhana yang meminta input dua bilangan dari user, kemudian menampilkan hasil dari semua operasi aritmatika dasar (penjumlahan, pengurangan, perkalian, dan pembagian) sekaligus.

‚úÖ Spesifikasi Program

  • Program meminta input dua bilangan dari user
  • Program menampilkan hasil penjumlahan, pengurangan, perkalian, dan pembagian
  • Gunakan format output yang sesuai dengan contoh
  • Pastikan pembagian menghasilkan bilangan bulat

üí° Contoh Output yang Diharapkan

=== KALKULATOR SEDERHANA ===
Masukkan bilangan pertama: 20
Masukkan bilangan kedua: 5

--- HASIL PERHITUNGAN ---
Hasil Penjumlahan: 25
Hasil Pengurangan: 15
Hasil Perkalian: 100
Hasil Pembagian: 4

üí≠ Tips Pengerjaan

  • Gunakan tipe data int untuk bilangan bulat
  • Gunakan operator aritmatika: +, -, *, /
  • Perhatikan format output yang diminta
  • Jangan lupa include <iostream>

üìù Contoh Struktur Program

#include <iostream>
using namespace std;

int main() {
    int bilangan1, bilangan2;
    
    // Header program
    cout << "=== KALKULATOR SEDERHANA ===" << endl;
    
    // Input dari user
    cout << "Masukkan bilangan pertama: ";
    cin >> bilangan1;
    cout << "Masukkan bilangan kedua: ";
    cin >> bilangan2;
    
    // Baris kosong
    cout << endl;
    
    // Header hasil
    cout << "--- HASIL PERHITUNGAN ---" << endl;
    
    // Tampilkan hasil operasi
    cout << "Hasil Penjumlahan: " << (bilangan1 + bilangan2) << endl;
    cout << "Hasil Pengurangan: " << (bilangan1 - bilangan2) << endl;
    cout << "Hasil Perkalian: " << (bilangan1 * bilangan2) << endl;
    cout << "Hasil Pembagian: " << (bilangan1 / bilangan2) << endl;
    
    return 0;
}

‚è∞ Soal 2: Program Konversi Detik

üìã Deskripsi Tugas

Buatlah program yang dapat mengkonversi total detik menjadi format jam, menit, dan detik. Program akan meminta input total detik dari user, kemudian menampilkan hasil konversi dalam format yang mudah dibaca.

‚úÖ Spesifikasi Program

  • Program meminta input total detik dari user
  • Program menghitung jam, menit, dan sisa detik
  • Program menampilkan hasil dalam format "X Jam, Y Menit, Z Detik"
  • Gunakan operator pembagian (/) dan modulus (%) untuk perhitungan

üí° Contoh Output yang Diharapkan

=== PROGRAM KONVERSI DETIK ===
Masukkan total detik: 3661

Hasil Konversi:
1 Jam, 1 Menit, 1 Detik

üßÆ Rumus Perhitungan

jam = total_detik / 3600 (1 jam = 3600 detik)
menit = (total_detik % 3600) / 60 (sisa detik dibagi 60)
detik = total_detik % 60 (sisa detik setelah menit)

üí≠ Tips Pengerjaan

  • Gunakan tipe data int untuk semua variabel
  • Manfaatkan operator pembagian (/) untuk mendapatkan hasil bagi
  • Manfaatkan operator modulus (%) untuk mendapatkan sisa bagi
  • Ingat: 1 jam = 3600 detik, 1 menit = 60 detik

üìù Contoh Struktur Program

#include <iostream>
using namespace std;

int main() {
    int total_detik, jam, menit, detik;
    
    // Header program
    cout << "=== PROGRAM KONVERSI DETIK ===" << endl;
    
    // Input dari user
    cout << "Masukkan total detik: ";
    cin >> total_detik;
    
    // Perhitungan konversi
    jam = total_detik / 3600;                    // 1 jam = 3600 detik
    menit = (total_detik % 3600) / 60;         // Sisa detik dibagi 60
    detik = total_detik % 60;                   // Sisa detik setelah menit
    
    // Baris kosong
    cout << endl;
    
    // Tampilkan hasil
    cout << "Hasil Konversi:" << endl;
    cout << jam << " Jam, " << menit << " Menit, " << detik << " Detik" << endl;
    
    return 0;
}
