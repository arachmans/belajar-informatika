Cara Membuat dan Menjalankan Fungsi

Apa itu Fungsi?

Fungsi adalah blok kode yang terorganisir dan diberi nama untuk melakukan tugas tertentu. Dalam matematika, fungsi memetakan input ke output, dan konsep serupa berlaku dalam pemrograman.

Analogi Sederhana: Fungsi itu seperti mesin kopi:

  • • Input: Kopi bubuk, air, gula (parameter)
  • • Proses: Menggiling, memanaskan, mencampur (kode di dalam fungsi)
  • • Output: Kopi siap minum (return value)

void - Fungsi Tanpa Nilai Kembalian

void adalah tipe data khusus dalam C++ yang menunjukkan bahwa fungsi tidak mengembalikan nilai apapun. Fungsi void digunakan ketika kita hanya ingin menjalankan proses tertentu tanpa perlu hasil kembali.

Struktur Fungsi

Fungsi void tanpa parameter
void namaFungsi() {
    // Kode fungsi
    // Tidak ada return
}
hello.cpp
#include <iostream>
using namespace std;

// Deklarasi fungsi void
void sapa() {
    cout << "Halo, Selamat Belajar!" << endl;
    // Tidak ada return karena ini fungsi void
}

int main() {
    // Memanggil fungsi void
    sapa();
    sapa();
    return 0;
}

Output Program 

Halo, Selamat Belajar!
Halo, Selamat Belajar!

Parameter dalam Fungsi

Parameter adalah nilai yang dikirim ke fungsi saat dipanggil. Fungsi dapat menerima satu atau lebih parameter.

Struktur Fungsi dengan Parameter

Fungsi void dengan parameter
void namaFungsi(tipeData parameter1, tipeData parameter2) {
    // Kode fungsi
    // Gunakan parameter di dalam fungsi
    // Tidak ada return
}
parameter.cpp
#include <iostream>
#include <string>
using namespace std;

// Fungsi void dengan 2 parameter
void cetakNama(string nama, int umur) {
    cout << "Nama: " << nama << endl;
    cout << "Umur: " << umur << " tahun" << endl;
    // Tidak ada return karena ini fungsi void
}

int main() {
    // Memanggil dengan argumen
    cetakNama("Andi", 20);
    cetakNama("Budi", 22);
    return 0;
}

Output Program 

Nama: Andi
Umur: 20 tahun
Nama: Budi
Umur: 22 tahun

Tip: Parameter membuat fungsi lebih fleksibel karena dapat menerima nilai yang berbeda setiap kali dipanggil.

Fungsi dengan Nilai Kembalian

Fungsi dapat mengembalikan nilai menggunakan kata kunci return. Tipe data nilai kembalian harus sesuai dengan tipe return fungsi.

Struktur Fungsi dengan Nilai Kembalian

Fungsi dengan return value dan parameter
tipeData namaFungsi(tipeData parameter1, tipeData parameter2) {
    // Kode fungsi
    // Proses parameter
    return nilaiKembali;
}
return_value.cpp
#include <iostream>
using namespace std;

// Fungsi mengembalikan integer dengan parameter
int tambah(int a, int b) {
    return a + b;
}

// Fungsi mengembalikan double dengan parameter
double bagi(double a, double b) {
    return a / b;
}

int main() {
    int hasil1 = tambah(5, 3);
    double hasil2 = bagi(10, 2);
    
    cout << "5 + 3 = " << hasil1 << endl;
    cout << "10 / 2 = " << hasil2 << endl;
    
    return 0;
}

Output Program 

5 + 3 = 8
10 / 2 = 5

int faktorial(int n) {
  if (n == 0) return 1;

Menggunakan perulangan

Konsep Kunci Rekursif

Base Case

Kondisi berhenti yang mencegah rekursif tak terbatas. Tanpa base case, program akan crash!

Recursive Case

Bagian fungsi yang memanggil dirinya sendiri dengan parameter yang mendekati base case.

recursion.cpp
#include <iostream>
using namespace std;

// Fungsi rekursif untuk menghitung faktorial
int faktorial(int n) {
    // Base case: kondisi berhenti
    if (n == 0) {
        cout << "Base case tercapai: faktorial(0) = 1" << endl;
        return 1;
    }
    // Recursive case: memanggil diri sendiri
    else {
        cout << "Menghitung " << n << " * faktorial(" << (n-1) << ")" << endl;
        return n * faktorial(n - 1);
    }
}

int main() {
    int angka = 5;
    cout << "Menghitung faktorial dari " << angka << endl;
    cout << "--------------------------------" << endl;
    int hasil = faktorial(angka);
    cout << "--------------------------------" << endl;
    cout << "Hasil: " << angka << "! = " << hasil << endl;
    return 0;
}

Output Program 

Menghitung faktorial dari 5
--------------------------------
Menghitung 5 * faktorial(4)
Menghitung 4 * faktorial(3)
Menghitung 3 * faktorial(2)
Menghitung 2 * faktorial(1)
Menghitung 1 * faktorial(0)
Base case tercapai: faktorial(0) = 1
--------------------------------
Hasil: 5! = 120

Soal 2: Konversi Suhu

Deskripsi:

Buat program untuk konversi suhu dengan fungsi-fungsi berikut:

  • Fungsi celsiusToFahrenheit(double c)
  • Fungsi fahrenheitToCelsius(double f)
  • Fungsi celsiusToKelvin(double c)

Rumus konversi:
• Fahrenheit = (Celsius × 9/5) + 32
• Celsius = (Fahrenheit - 32) × 5/9
• Kelvin = Celsius + 273.15

Contoh Input:

Masukkan suhu dalam Celsius: 25

Contoh Output:

25 Celsius = 77 Fahrenheit
25 Celsius = 298.15 Kelvin
temperature.cpp

Tips Menyelesaikan Latihan:

  • Pahami terlebih dahulu rumus yang akan digunakan
  • Buat fungsi satu per satu dan test masing-masing
  • Perhatikan tipe data yang digunakan (int vs double)
  • Gunakan komentar untuk memudahkan debugging
