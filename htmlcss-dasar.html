üìù HTML (HyperText Markup Language)

Apa itu HTML?

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML menggunakan tag untuk menandai berbagai elemen dalam dokumen.

Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat datang di website saya!</h1>
    <p>Ini adalah paragraf pertama.</p>
</body>
</html>

üí° Poin Penting yang Harus Diingat

1

Tag Case-Insensitive: <P> sama dengan <p>, tapi gunakan huruf kecil untuk konsistensi.

2

Atribut Opsional: Tidak semua elemen butuh atribut, tapi beberapa atribut wajib (seperti src pada img).

3

Self-Closing Tags: Beberapa elemen seperti <img>, <br>, <hr> tidak memiliki tag penutup.

4

Nesting Rules: Elemen bisa berisi elemen lain, tapi harus ditutup dengan urutan yang benar.

üé® CSS (Cascading Style Sheets)

Apa itu CSS?

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan layout halaman web. CSS mengontrol warna, font, ukuran, posisi, dan aspek visual lainnya dari elemen HTML.

Cara Menambahkan CSS ke HTML

1. Inline CSS

<p style="color: blue; font-size: 18px;">Teks berwarna biru</p>

2. Internal CSS

<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

3. External CSS

<head>
    <link rel="stylesheet" href="style.css">
</head>

üéØ Class Selector

Memilih elemen berdasarkan atribut class. Satu class bisa digunakan berkali-kali.

Sintaks: .namaClass { properti: nilai; }
HTML:
<p class="highlight">Teks ini</p>
<div class="highlight">Div ini</div>
CSS:
.highlight {
    background: yellow;
    padding: 10px;
}
Hasil:

Teks ini memiliki class highlight

Div ini juga memiliki class highlight

Paragraf ini tidak memiliki class

üÜî ID Selector

Memilih elemen berdasarkan atribut ID. ID harus unik dalam satu halaman.

Sintaks: #namaID { properti: nilai; }
HTML:
<h1 id="judul-utama">Judul</h1>
<p id="paragraf-khusus">Paragraf</p>
CSS:
#judul-utama {
    color: red;
    text-align: center;
}
Hasil:

Judul dengan ID khusus

Paragraf biasa tanpa styling

Paragraf lain juga tidak terpengaruh

üåê Universal Selector

Memilih SEMUA elemen di halaman. Biasanya digunakan untuk reset CSS.

Sintaks: * { properti: nilai; }
CSS:
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
Penjelasan:

‚ö†Ô∏è Universal selector mempengaruhi SEMUA elemen, termasuk html, body, div, p, h1, dll. Contoh di atas akan menghilangkan margin dan padding default dari semua elemen.

Hati-hati: Penggunaan berlebihan dapat memperlambat website!

üìã Attribute Selector

Memilih elemen berdasarkan atribut tertentu yang dimilikinya.

Sintaks: [atribut="nilai"] { properti: nilai; }
HTML:
<input type="text" placeholder="Nama">
<input type="email" placeholder="Email">
<input type="password" placeholder="Password">
CSS:
input[type="email"] {
    border: 2px solid blue;
}
Hasil:


Hanya input dengan type="email" yang memiliki border biru
